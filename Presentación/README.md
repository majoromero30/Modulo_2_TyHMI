## Entrega Presentación 
Para la creación de las diapositivas empleamos R Markdown con el formato ioslides. Este flujo de trabajo nos permitió integrar directamente el código de R, los cálculos de tiempo real y los gráficos estadísticos en un solo documento dinámico. Al ejecutar el proceso de "tejido" (Knit), el sistema procesa el código fuente y genera automáticamente una presentación interactiva en formato HTML, asegurando que los resultados visualizados sean el reflejo exacto de los datos procesados, sin errores de transcripción manual.
