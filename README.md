# Modulo_2_TyHMI
Este repositorio documenta una serie de experimentos técnicos sobre eficiencia algorítmica y herramientas modernas de ingeniería. El trabajo se centra en el análisis empírico de la performance computacional, comparando métodos iterativos tradicionales frente a soluciones vectorizadas en R para optimizar el procesamiento de datos a escala industrial.

Para el desarrollo de estos ejercicios se utilizó Posit Cloud y programación literaria en R Markdown.
